A collection of JavaScript fundamentals exercises and projects 
built during Week 4. Covers variables, data types, functions, 
control flow, arrays, objects and mini projects.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Features
- Task 7.1: Variables and data types practice
- Task 7.2: Operators and expressions
- Task 7.3: Functions - declarations, expressions and arrow functions
- Task 7.4: Control flow - if/else, switch and loops
- Task 8.1: Arrays and array methods
- Task 8.2: Objects and object methods
- Task 8.3: Arrays of objects
- Mini Project 1: Interactive Calculator
- Mini Project 2: Student Grade Tracker

## How to Run
1. Clone this repository
2. Open any task folder
3. Open `index.html` in your browser
4. Press `F12` and click Console tab to see outputs

## Lessons Learned
- How to declare and use variables with let and const
- How to work with different data types in JavaScript
- How to write and call functions in different ways
- How to control program flow using if/else and loops
- How to store and manipulate data using arrays and objects
- How to combine arrays and objects for real world data
